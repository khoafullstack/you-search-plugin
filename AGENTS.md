# AGENTS.md

## Project Description

The main idea of this project is creating 2 tools which called `web_you_fetch` and `web_you_search` to use You.com Platform.

## Tech:

- Program Lanuage: Javascript (Built Bundle)/TypeScript
- Env: pnpm

## Folder Structure:

```
.
├── shared/              # Core logic for both extensions
├── pi-extension/        # Extension for Pi coding agent
├── opencode-plugin/     # Plugin for Opencode
├── openspec/            # Specs & config for You.com integration
└── package.json
```
